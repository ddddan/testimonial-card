# Testimonial Card

A responsive testimonial card component built as part of the GreatFrontend Projects challenge. This project showcases a clean, modern design for displaying customer testimonials with profile information.

## Description

This is a simple web component that displays a testimonial card featuring:
- A profile thumbnail image
- User's name and username
- A testimonial quote

The design is responsive and works across desktop, tablet, and mobile devices.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile viewports
- **Clean UI**: Modern card-based layout with subtle shadows and rounded corners
- **Typography**: Uses Noto Sans font for readability
- **Accessibility**: Proper semantic HTML and alt text for images

## Technologies Used

- **HTML5**: Semantic markup for the testimonial structure
- **CSS3**: Custom styling with flexbox for layout and responsive design
- **Google Fonts**: Noto Sans font family

## Installation & Setup

1. Clone or download this repository
2. Open `index.html` in your web browser, or
3. Serve the files using a local server:

   ```bash
   python -m http.server 8000
   ```

   Then navigate to `http://localhost:8000` in your browser.

## Usage

Simply open the `index.html` file in any modern web browser. The testimonial card will be displayed with the sample content.

The site is also hosted at `https://ddddan.github.io/testimonial-card`
